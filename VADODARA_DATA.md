# Vadodara Dummy Data for Admin Panel

Use this data to populate the **Zone** and **Area** forms in the Admin Panel.

## 1. Create Zone (City)

Go to **Hierarchy > Zones** and click **+ Add New Zone**.

| Field | Value |
|-------|-------|
| **Country** | India |
| **State** | Gujarat |
| **City** | Vadodara |
| **Max Areas** | 20 |
